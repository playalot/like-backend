##Notification

消息中心列表

######URL

http://api.likeorz.com/v1/notification/{page:[0-9]+}

######支持格式
JSON

######HTTP请求方式

GET

######请求参数
|参数|必选|类型及范围|说明|
|--------|-------|-------|
|LIKE-SESSION-TOKEN|true|string|Header中带用户授权|
|{page:[0-9]+}|true|int|分页|

######返回字段说明
|返回值字段|字段类型|字段说明|
|--------|-------|-------|
|notes|array|消息列表|
|type|string|消息类型(LIKE MARK FOLLOW)|
|num|string|消息数量|
|is_new|int|是否是新消息|
|updated|int|消息产生时间|
|user|array|用户信息|
|user_id|int|用户ID|
|nickname|string|用户昵称|
|avatar|string|用户头像|
|posts|array|作品信息|
|content|string|作品缩略图|
|next|int|下一页页号（没有下一页返回 false）|
